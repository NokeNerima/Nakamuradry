　Javardry HTML5版は、Java製のWizクローンであるJavardryをHTML5
（HTML5 + CSS + JavaScript）に移植したものです。

　実行するには、Javardry.html（コントローラ無し、PC用）か
JavardryController.html（コントローラ有り、スマホ用）を
ブラウザで読み込んで下さい。末尾に_engとついているファイルは
英語版です。

　ローカル（オフライン）でもオンライン（ネット上にアップしたもの）でも
動作しますが、ローカル動作時にはダンジョンの画像表示が出来ないため、
オンライン動作（ローカルサーバでの動作を含む）をお薦めします。

※キャンプ中、パーティー内でのアイテム受け渡し時、アイテムを渡す対象者は
「左右キー」ではなく「左右移動キー」で切り替えられます。

　詳しくはマニュアルを見て下さい。
