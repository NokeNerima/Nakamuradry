  Javardry HTML5 Edition is a port of Javardry, a Wiz clone made in Java, to HTML5(HTML5 + CSS + JavaScript). It was originally created in Japanese and has been machine translated. See the Japanese manual for details.

  To execute, load the Javardry_eng.html file (without controller, for PC) or the JavardryController_eng.html file (with controller, for smartphone) in your browser.

  It can run both locally (offline) and online (uploaded on the Internet), but online operation (including operation on a local server) is recommended because dungeon images cannot be displayed when running locally.
